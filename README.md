# satellite-fire-risk-analytics
This project analyses near real-time NASA FIRMS VIIRS satellite fire data using Tableau and Python. The objective is to identify global fire hotspot distribution, fire intensity, cofidence levels and temporal detection patterns through interactice dashboarda and storytelling.

Dataset
| Item            | Value                     |
| --------------- | ------------------------- |
| Source          | NASA FIRMS                |
| Records         | 220,679                   |
| Original Fields | 13                        |
| Final Fields    | 23                        |
| Period          | 03 May 2026 – 10 May 2026 |
